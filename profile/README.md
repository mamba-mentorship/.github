# Black Mamba

Black Mamba is an aggressive mentorship for non-beginners.

## It aims to cover the following topics:

- Data Structure
- Design Patterns
- Software Architecture
- Cloud Computing
- Infrastructure
- Automation
- Virtualization / Containerization
- Service Mesh
- Observability
- Security
- Scalability
- Governance
- Performance optimization
- And more ...
